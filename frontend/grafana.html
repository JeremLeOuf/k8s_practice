📊 Access Grafana Dashboard

Grafana runs in your Kubernetes cluster. To access it:

🚀 Quick Start

Run this command in your terminal:

kubectl port-forward -n grafana svc/grafana-service 3000:3000

Or use the helper script: ./scripts/access-grafana.sh

💡 What happens next?

  1. The port-forward command will start (keep it running)
  2. Grafana will be accessible at http://localhost:3000
  3. Login with: admin / admin

⚠️ Note

The Grafana interface cannot be embedded in this CDN page for security reasons. You must access it directly at http://localhost:3000 after setting up port-forward.
